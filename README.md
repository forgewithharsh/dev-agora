# DevAgora

DevAgora is a modern developer networking platform designed to foster meaningful technical discussions, collaboration, and connections within the developer community.

## Overview

DevAgora enables developers to discover, connect, and engage with like-minded individuals through an intuitive and interactive experience. Inspired by swipe-based discovery, the platform focuses on professional networking rather than casual interaction.

## Features

- Developer discovery (interest / ignore)
- Profile-based matching system
- Connection requests and matches
- Clean and minimal developer-focused UI

## Tech Stack

**Frontend**
- React (Vite)

**Backend**
- Node.js
- Express.js
- MongoDB

## Core Concepts

- RESTful API architecture
- JWT-based authentication
- Scalable and modular backend structure
- Optimized MongoDB schema design

## Project Goal

To build a high-quality developer ecosystem where individuals can connect, share knowledge, and collaborate on meaningful projects.
